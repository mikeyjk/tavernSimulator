addEntity function

This function adds an entity to the gameWorld class The string determines what entity you can add, the \ different types of entities are: NPC, Terrain, Geometry, \ Character, OtherPlayer. These entities are added to the GameWorld maps and are then \ have thier update called every frame. Also this function adds the models and textures to the Resource Manager, \ if they havent previously been added.
