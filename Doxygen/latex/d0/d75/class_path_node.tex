\subsection{Detailed Description}
\doxyref{Path\+Node}{p.}{d0/d75/class_path_node} class represents a position on the world map. 

\doxyref{Path\+Node}{p.}{d0/d75/class_path_node} class represents a position on the world map represented by an x, y, and z coordinate. Also stores cost to travel to the node and heuristic distance to target.
