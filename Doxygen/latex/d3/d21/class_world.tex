add\+Entity function 

This function adds an entity to the game\+World class The string determines what entity you can add, the \textbackslash{} different types of entities are\+: \doxyref{N\+P\+C}{p.}{d0/d2a/class_n_p_c}, Terrain, \doxyref{Geometry}{p.}{db/d0b/class_geometry}, \textbackslash{} \doxyref{Character}{p.}{d7/da7/class_character}, Other\+Player. These entities are added to the Game\+World maps and are then \textbackslash{} have thier update called every frame. Also this function adds the models and textures to the \doxyref{Resource}{p.}{df/d00/class_resource} Manager, \textbackslash{} if they havent previously been added.
