Simple \doxyref{Open\+G\+L}{p.}{d8/d24/class_open_g_l} Image Library

A tiny c library for uploading images as textures into \doxyref{Open\+G\+L}{p.}{d8/d24/class_open_g_l}. Also saving and loading of images is supported.

I'm using Sean's Tool Box image loader as a base\+: {\tt http\+://www.\+nothings.\+org/}

I'm upgrading it to load T\+G\+A and D\+D\+S files, and a direct path for loading D\+D\+S files straight into \doxyref{Open\+G\+L}{p.}{d8/d24/class_open_g_l} textures, when applicable.
